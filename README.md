## Start the server

This command serves the REST data at `http://localhost:3001`:

    npm start

This provides you following routes:

Method | Route              | Result
------ | ------------------ | ------
GET    | `/api/peoples`     | List all peoples
GET    | `/api/peoples/:id` | Get details about `:id`
PUT    | `/api/peoples/:id` | Update `:id`
